# Port scanner in Go
Go-lang script for scanning specific or all open ports of the device.
# Packages Used
- fmt
- time
- sync
- strings
- net

# How to run
Navigate to the project directory.
```bash
cd Rotten-Scripts/Go/Port_Scanner
```
```bash
go run port_scanner.go 
```
