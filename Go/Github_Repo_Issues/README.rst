Github-Repo-Issues
==================

|checkout|

This is Go script used to check for the Issues for any Repo .

Prerequisites:
--------------

Go

Process:
--------

Replace HarshCasper by UserName-of-Github and Rotten-Scripts by RepoName in main.go const issueurl

.. code-block:: bash

    $ go run main.go
