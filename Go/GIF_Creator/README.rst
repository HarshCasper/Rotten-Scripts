Video/Images to GIF converter in golang
=======================================

|checkout|

Use `ffmpeg <https://ffmpeg.org/>`__ before using script
--------------------------------------------------------

Steps to run:
-------------

.. code-block:: bash

   git clone
   go install gifgen.go
   gifgen -s (image/video) -i (input path for images directory) - o (output path for generated gif)
