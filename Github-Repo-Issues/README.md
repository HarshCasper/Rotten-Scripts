# Github-Repo-Issues  
This is Go script used to check for the Issues for any Repo .

### Prerequisites:  
Go

### Process:
 ```go
  Replace HarshCasper by UserName-of-Github
  andRotten-Scripts by RepoName in main.go const issueurl 
  $ go run main.go
  ``` 
