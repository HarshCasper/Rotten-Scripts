TSV to CSV Converter
====================

|checkout|

Description
-----------

This Scripts converts the TSV (Tab-separated values) to CSV
(Comma-separated values)

Execution
---------

-  Run ``python TSV_to_CSV.py``
-  Provide the File Path of TSV file
-  Task Done!
