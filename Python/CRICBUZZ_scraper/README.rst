Cricbuzz scrapper
=================

|checkout|

This python script will scrap `cricbuzz <www.cricbuzz.com>`__ to get
live scores.

Technologies used:
------------------

-  urllib.request
-  BeautifulSoup

Run the script :
----------------

.. code:: sh

   $ python3 cric_buzz.py
