# Email Scraper for commit history

A simple python script to crawl all email address, name and username from a specific repository on GitHub.

## Setup instructions

| Install:          |
| ----------------- |
| `bash install.sh` |

## Run instructions

| Run:                                    |
| --------------------------------------- |
| `Python email_scrape.py --help`         |
| `python email_scrape.py <token> <repo>` |
