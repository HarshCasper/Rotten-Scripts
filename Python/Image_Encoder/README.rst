Image Encoder
=============

It is a Image Automation tool that is used to encode and decode the
image.

Usage
-----

Image Encode
------------

.. code-block:: bash

  python3 image_encoder.py -ie "Image_Path"

Image Decode
------------

.. code-block:: bash

  python3 image_encoder.py -id "Image_Path" -k "Key_value"
