Running the bot
---------------

.. code-block:: bash

   # install python3 and python3-pip if they are not installed

   # install the requirements
   $ pip3 install discord.py

open package.json file and paste your token and prefix

.. code-block:: json
 
   {
       "prefix": "Token HERE",
       "token" : "PREFIX HERE"
   }

.. code:: bash

   #run main file
   $ python3 main.py
