Scrap Top Movies from IMDB
==========================

|checkout|

This Python script will Scrap the details of top 250 movies from IMDB
and generate a CSV file for it

Technologies used:
------------------

-  pandas
-  BeautifulSoup

Running the script:
-------------------

.. code:: sh

   $ python3 imdb_top.py
