# Linkedin Scrapper

Script scraps the data of Linkedin Connections(first) of a User on Linkedin

It scraps the following data on All Connections
- Name
- Linkedin_URL
- Latest_JOB_Position
- Skills


## Requirements

- Linkedin Profile
- Python Setup on System
- Chromium
- Chrome
- Install Selenium
- Install bs4 (```pip3 install bs4```)


## How to Run

``` python3 main.py```
