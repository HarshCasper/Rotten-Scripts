### Get time in different timezones/ countries

##### Using the script
*  With country code
    ```
    python script.py --country <countryCode>
    ```
    Eg:
    ```
    python script.py --country in
    ```

* With location
    ```
    python script.py --location <location>
    ```
    Eg:
    ```
    python script.py --location 'Asia/Kolkata'
    ```
